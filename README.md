# 🎬 Movie Genre Generation Using Machine Learning

## Overview

Movie Genre Generation is a Machine Learning project that predicts movie genres from movie descriptions, plot summaries, or other textual information. The system uses Natural Language Processing (NLP) and Machine Learning algorithms to analyze text and classify movies into appropriate genres.

## Features

* Predicts movie genres automatically
* Text preprocessing and cleaning
* NLP-based feature extraction
* Machine Learning classification model
* Easy-to-use interface
* Fast and accurate predictions
* Supports multiple movie categories

## Project Objectives

* Analyze movie descriptions using NLP techniques
* Extract meaningful features from text
* Train a machine learning model for genre prediction
* Improve movie categorization and recommendation systems

## Technologies Used

* Python
* Machine Learning
* Natural Language Processing (NLP)
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Dataset

The project uses a movie dataset containing:

* Movie Title
* Genre
* Plot Summary
* Keywords
* Additional Movie Information

## Workflow

1. Data Collection
2. Data Preprocessing
3. Text Cleaning
4. Feature Extraction
5. Model Training
6. Genre Prediction
7. Performance Evaluation

## Machine Learning Process

### Data Preprocessing

* Remove unwanted characters
* Convert text to lowercase
* Remove stop words
* Tokenization

### Feature Extraction

* TF-IDF Vectorization
* Bag of Words

### Model Training

* Train machine learning models
* Evaluate model accuracy
* Select the best-performing model

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/movie-genre-generation.git
```

2. Navigate to the project folder

```bash
cd movie-genre-generation
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python app.py
```
